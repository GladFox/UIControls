# Product Context

## Цель
Создать библиотеку UI-контролов на базе Unity `uGUI` с анимациями через `DOTween`.

## Продуктовые принципы
- Переиспользуемые контролы с единым подходом к анимации.
- Предсказуемое состояние и явный API для вызова из кода.
- Минимальная связность между контролами.
- Внешние UI-решения используются только как источник идей, но не как runtime-зависимость.

## Текущий фокус
- Первая итерация runtime-библиотеки (`Button`, `Toggle`, `Modal`, `ProgressBar`).
- Статичный demo-поток: сцена/префабы настраиваются в редакторе через сериализуемые поля, без runtime-создания UI-структуры.
- Переиспользуемые `ScriptableObject`-пресеты визуальных состояний и расширяемые SO-действия для пользовательской логики.
